# 🤖 Telegram Business Bot — Установка и запуск

## Шаг 1: Создать бота в Telegram
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Придумайте имя (например: **Цифровое Агентство**)
4. Придумайте username (например: `myagency_bot`)
5. Скопируйте **API токен** (вида `123456789:ABCdefGHI...`)

## Шаг 2: Установить Python
- Скачайте Python 3.10+ с [python.org](https://python.org)
- При установке поставьте галочку ✅ "Add Python to PATH"

## Шаг 3: Скачать файлы бота
- Все файлы бота должны быть в одной папке:
  - `bot.py` — основной код
  - `config.py` — настройки (токен, контакты)
  - `requirements.txt` — зависимости

## Шаг 4: Установить зависимости
Откройте терминал (командную строку) в папке с ботом и выполните:
```
pip install -r requirements.txt
```

## Шаг 5: Вставить свой токен
Откройте файл `config.py` и вставьте:
- `BOT_TOKEN` — токен от BotFather
- `YOUR_TELEGRAM` — ваш Telegram username (без @)
- `YOUR_PHONE` — ваш номер телефона

## Шаг 6: Запустить бота
В терминале в папке с ботом:
```
python bot.py
```

## Шаг 7: Настроить меню бота
В @BotFather отправьте:
- `/setmenubutton` → выберите бота → отправьте текст: `📋 Меню`

## Круглосуточная работа (опционально)
### Вариант 1: Хостинг (рекомендуется)
- [Amvera.cloud](https://amvera.cloud) — бесплатный запуск
- [Railway.app](https://railway.app) — простой деплой
- [PythonAnywhere](https://pythonanywhere.com)

### Вариант 2: Своё VPS
1. Арендуйте VPS (от 3-5$ в месяц)
2. Загрузите файлы бота
3. Запустите: `nohup python3 bot.py &`

### Вариант 3: Локально (ПК должен быть включён)
Просто запустите `python bot.py` — бот работает, пока включён компьютер.
